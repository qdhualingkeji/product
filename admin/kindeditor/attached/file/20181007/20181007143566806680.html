杭州通过规定：大气重污染II级预警时不得施工某建设项目建设期2年在建设期初设备资金100万六(1)班在一次数学测试中，全班58人有8人成绩没有达到优秀。这测试数学成绩的优秀率是多少？关于尊老爱幼的经典故事

转自：百度作业帮百科新华网中科院图书馆

昨天上午，代市长张鸿铭主持召开市政府第18次常务会议。

会议学习贯彻落实市委十一届六次全会精神，研究并原则通过《政府工作报告(送审稿)》《关于杭州市2013年国民经济和社会发展计划执行情况与2014年国民经济和社会发展计划草案的报告(送审稿)》和《关于杭州市及市本级2013年财政预算执行情况和2014年财政预算草案的报告(送审稿)》。

昨天会上说，将认真研究吸纳收集到的意见建议，再作修改完善，力求形成一个符合杭州实际、顺应人民群众新期待的报告，努力推进杭州改革发展不断取得新突破。

以上“三大报告”，将提交2月10日召开的市十二届人大四次会议上审议。

会议强调，政府各部门要把思想和行动统一到市委决策上来，切实把市委全会精神和《决定》要求体现到今年各项工作中，体现到政府自身建设中，体现到政府机关干部队伍建设中，在解放思想上、在政府自身改革上、在经济转型升级上、在统筹城乡发展上、在美丽杭州建设上、在社会事业改革上抓落实、求突破，努力开创政府工作新局面。

昨天的会议还研究并原则通过了《杭州市建设工程文明施工管理规定》(下简称“规定”)和《杭州市建筑市场管理若干规定》。

《规定》对建筑施工扬尘的控制，作出了更严格的规定。

譬如，第十九条规定，建筑材料易产生扬尘的，应当进行喷淋、遮盖处理。

第二十一条规定，天气预报风速五级以上或者发布大气重污染二级预警时，不得进行建(构)筑物拆除施工和土地平整、换土、原土过筛等作业，并应当对施工现场采取喷淋、覆盖等降尘措施。

如果未按照规定采取防止扬尘措施的，将处2000元以上1万元以下的罚款。
